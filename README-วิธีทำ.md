# PumpDesk — ไฟล์สำหรับขึ้นระบบ + วิธีทำ

ในโฟลเดอร์นี้มี 4 ไฟล์ที่ต้องใช้:

| ไฟล์ | คืออะไร |
|---|---|
| `index.html` | หน้าหลังบ้าน (แดชบอร์ด/ออเดอร์/ลูกค้า) — เปิดในคอมได้เลย |
| `order.html` | หน้าใส่ออเดอร์บนมือถือ (Mini App ใน Telegram) |
| `support.js` | ตัวช่วยที่ `order.html` ต้องใช้ — ต้องวางไว้โฟลเดอร์เดียวกัน |
| `bot.py` | บอท Telegram (ถ้าอยากให้กด /order แล้วเด้งฟอร์ม) |

> ทั้งหมดเชื่อมกันผ่าน Firebase (ตั้งค่าไว้ในไฟล์แล้ว) ออเดอร์ที่ใส่จากมือถือจะขึ้นแดชบอร์ดบนคอมแบบเรียลไทม์

---

## 1) ขึ้นเว็บ (deploy)

ต้องเป็น **https** เท่านั้น (Telegram Mini App บังคับ) — แนะนำ render.com / Netlify / Vercel

1. เอาไฟล์ `index.html`, `order.html`, `support.js` อัปขึ้นโฮสต์ (ไว้โฟลเดอร์เดียวกัน)
2. เปิด `https://โดเมนของคุณ/` → เจอหน้าหลังบ้าน สมัคร/ล็อกอินได้เลย
3. หน้าใส่ออเดอร์อยู่ที่ `https://โดเมนของคุณ/order.html`

> ถ้าใช้ render.com เดิมอยู่แล้ว แค่ push 3 ไฟล์นี้ทับของเก่า

---

## 2) ตั้งปุ่มใส่ออเดอร์ใน Telegram (Mini App)

1. เปิด **@BotFather** → `/newbot` (ถ้ายังไม่มีบอท) เก็บ **Bot Token** ไว้
2. ที่ @BotFather → `/setmenubutton` → เลือกบอท → วาง URL:
   ```
   https://โดเมนของคุณ/order.html?shop=ชื่อร้านของคุณ
   ```
   (`ชื่อร้าน` = ชื่อผู้ใช้ที่ล็อกอินหน้าหลังบ้าน เพื่อให้ออเดอร์เข้าร้านถูกคน)
3. ตั้งชื่อปุ่ม เช่น `➕ ใส่ออเดอร์`
4. เสร็จ! ในแชตบอทจะมีปุ่มเมนู กดแล้วฟอร์มเด้งขึ้น กรอกเสร็จออเดอร์เข้าแดชบอร์ดทันที

> ลิ้งนี้ก็อปได้จากในหน้าหลังบ้าน → ปุ่ม "เชื่อม Telegram" → หัวข้อ "ปุ่มเมนูใส่ออเดอร์"

---

## 3) (ถ้าต้องการ) รันบอท bot.py — ให้พิมพ์ /order แล้วเด้งฟอร์ม + สรุปกลับแชต

1. ติดตั้ง Python 3.10+ แล้วรัน:
   ```
   pip install "python-telegram-bot==21.*"
   ```
2. เปิด `bot.py` แก้ 2 บรรทัด:
   - `BOT_TOKEN` = โทเคนจาก @BotFather
   - `WEBAPP_URL` = `https://โดเมนของคุณ/order.html?shop=ชื่อร้านของคุณ`
3. รัน:
   ```
   python bot.py
   ```

---

## 4) แจ้งเตือนเข้า Telegram เวลาปิดยอด (ไม่บังคับ)

ในหน้าหลังบ้าน → ปุ่ม **"เชื่อม Telegram"** → ใส่ Bot Token + Chat ID (มีปุ่ม "ดึงอัตโนมัติ")
พอกดสถานะออเดอร์เป็น **ทำแล้ว** ระบบจะส่งสรุป (รวมประเภทแอคเค้า) เข้าแชตให้อัตโนมัติ

---

## อัปเดตล่าสุดในเวอร์ชันนี้
- ✅ ข้อมูลไม่หายแล้ว — ออกจากหน้าแล้วกลับเข้ามาเห็นออเดอร์ทันที ไม่ต้องรีเฟรช
- ✅ หน้าออเดอร์แสดง **เวลา** ที่เข้าออเดอร์
- ✅ มีปุ่ม **คัดลอกลิ้ง** ในแต่ละออเดอร์
- ✅ เลือก **ประเภทแอคเค้า** (ปกติ/ปน หรือ คนไทย) ตอนใส่ออเดอร์ และโชว์ในหน้าออเดอร์
- ✅ ปุ่ม **วางลิ้ง** ในฟอร์มมือถือใช้งานได้ใน Telegram แล้ว
